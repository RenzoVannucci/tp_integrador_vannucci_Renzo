Acerca del Sistema

Estudiante: Vannucci Renzo

Este sistema permite administrar estudiantes mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar) utilizando una base de datos PostgreSQL.

Información del sistema

  • Nombre: Sistema de Gestión de Estudiantes
  • Versión: 1.0
  • Tipo: Aplicación web CRUD
  • Base de datos: PostgreSQL

Funcionalidades implementadas

  • Alta de estudiantes
  • Modificación de estudiantes
  • Eliminación de estudiantes
  • Búsqueda por DNI, apellido, nombre y carrera
  • Validación de datos ingresados
  • Mensajes de éxito y error mediante Flash
  • Gestión de teléfono y edad de estudiantes

Tecnologías utilizadas

  • Python 3
  • Flask
  • PostgreSQL
  • psycopg2
  • HTML5
  • CSS3
  • Jinja2

Proyecto desarrollado con fines educativos para practicar desarrollo backend, manejo de bases de datos relacionales y aplicaciones web con Flask.

Arquitectura

  • Frontend: HTML + CSS
  • Backend: Flask
  • Persistencia: PostgreSQL
  • Acceso a datos: psycopg2
